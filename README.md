# Log Analyzer

Этот скрипт анализирует файлы логов Apache с заданным форматом и собирает статистику о запросах.

## Формат лога

Логи должны соответствовать следующему формату:

%h - - %t "%r" %s %b "%{Referer}" "%{User-Agent}" %d

## Возможности

- Обработка одного файла или всей директории.
- Сбор следующей статистики:
  - Общее количество запросов
  - Количество запросов по HTTP-методам
  - Топ 3 IP-адресов
  - Топ 3 самых долгих запросов
- Сохранение результата в JSON
- Вывод результата в терминал

## Как использовать

### Установка

Ничего устанавливать не нужно. Скрипт написан на чистом Python 3.

### Запуск

```bash
python analyzer.py ./logs/
```
или

```bash
python analyzer.py ./logs/access.log
```

Результат
Результат будет выведен в терминал и сохранён в файл с расширением .json рядом с каждым лог-файлом.
